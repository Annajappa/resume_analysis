# ResumeAnalyzer

**ResumeAnalyzer** is a full-stack web application that allows users to upload and analyze resumes efficiently. It extracts key information from resumes, such as personal details, skills, education, and experience, providing valuable insights for recruiters and job seekers alike.  

---

## Features
- Upload resumes in multiple formats (PDF, DOCX).  
- Automatic extraction of key information (skills, experience, education).  
- Interactive dashboard to view analyzed data.  
- Search and filter resumes based on skills or experience.  
- Responsive design for desktop and mobile devices.  

---

## Tech Stack
- **Frontend:** React.js, Material-UI  
- **Backend:** Node.js, Express.js  
- **Database:**  SQLite  
- **Others:** REST API integration, file upload handling
